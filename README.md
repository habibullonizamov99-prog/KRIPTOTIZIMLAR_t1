# KRIPTOTIZIMLAR_t1
TOPSHIRIQ


OpenSSL yordamida TLS sertifikat yaratish, operatsion tizimning ishonchli sertifikatlar omboriga qo‘shish va lokal tarmoqda himoyalangan veb-tizimni joriy etish. 
Maqsad • Lokal tarmoqda ishlaydigan xavfsiz veb-tizimni yaratish; • O‘zining sertifikatlash markazi (Root CA) ni tashkil etish; • Server va foydalanuvchi uchun alohida ochiq va shaxsiy kalitlar yaratish; • Sertifikatlarni operatsion tizimning ishonchli (trusted) root sertifikatlar ro‘yxati ga qo‘shish; • Saytga kirishda login va parol orqali identifikatsiya qilish; • Foydalanuvchilarga rol berish va ruxsatlarni cheklash; • Parollarni zamonaviy talablar asosida tekshirish va Argon2id algoritmi yordamida saqlash; • Server va mijoz o‘rtasida zamonaviy usulda umumiy kalit hosil qilish hamda ma’lumotlarni AES-256-GCM rejimida shifrlash; • Barcha jarayonlarni skrinshotlar bilan hujjatlashtirish.
